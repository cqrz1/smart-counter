# Smart Counter

A modern and interactive counter application built with Vanilla JavaScript, Tailwind CSS, and Vite.

## Features

- Increase Counter
- Decrease Counter
- Reset Counter
- Random Number Generator
- Maximum Limit (100)
- Minimum Limit (0)
- Total Increments Counter
- Total Decrements Counter
- Dynamic Status Messages
- Progress Tracking
- Recent Activity Section
- Responsive Design (Mobile, Tablet, Desktop)
- Clean UI with Tailwind CSS

## Built With

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- Vite
